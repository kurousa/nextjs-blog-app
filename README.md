# React x Next.js による ブログアプリ

## 概要

- ローカルマークダウンファイル読み込みに対応した記事一覧、記事詳細ページの表示ができます。
  - マークダウンファイルパーサーとして、gray-matter, remark, remark-htmlを利用しています。

## 技術スタック

- React v19.2.1
- Next.js(AppRouter) v16.0.8 /w ReactCompiler
- TypeScript v5
- Tailwind CSS v4
- Node.js v22
- Markdown
  - gray-matter
  - remark
  - remark-html
- Biome(Linter/Formatter) v2.2.0
